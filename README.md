# Simple JS accordion

### App demo
App is available at:
https://accordion-js.herokuapp.com/

### Installation

Requires [Node.js](https://nodejs.org/) v4+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ yarn install
```

For develop environment...

```sh
$ yarn run dev-server
```

### Comands

For creat dist build
```sh
$ yarn run build
```
